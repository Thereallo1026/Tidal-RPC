# Tidal-RPC
A Python Program that displays Current Tidal Song info &amp; name in your Discord status with Rich Presence

## Requirements
```
Python 3 or newer
Discord
```
## Usage
```
git clone https://github.com/Thereallo1026/Tidal-RPC
pip install -r requirements.txt 
py tidal.py
```
